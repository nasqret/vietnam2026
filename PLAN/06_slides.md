# Module 06 — Slides (reveal.js, one deck per lecture)

**Goal:** a presentation deck per lecture to follow the talk live — reveal.js via Markdown, math-first
theme, with links back into the book and the Lambda Lab. Mirrors the `slides_*.html` approach of
`classical-foundations-ann`.

## Subtasks
- [x] `slides/index.html` deck picker + shared theme/CSS.
- [ ] `slides/lecture1..6.html` — 10–14 slides each: motivation, core, a live-lab moment, a proof, recap.
- [ ] Speaker notes; PDF export via `decktape`/print.
- [ ] Cross-link every deck to its book chapter and any lab command used.

## Acceptance criteria
- Each deck opens standalone, advances with keyboard, and renders math.
- Deployed under `/vietnam2026/slides/`.
